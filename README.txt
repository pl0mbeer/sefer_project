Проект представляет из себя рабочий продукт для переименования файла текстом на картинке.
На вход подается путь к папке с фотографиями. Далее можно написать префикс для переименования. Затем обученная модель на базе yolo11 детектирует табличку с текстом. После этого делается кроп детектированной области и предобработка. Далее полученный кроп сканируется собственной обученной моделью с архитектурой CRNN. После этого файл переименовывается и результат заносится в таблицу, как и уверенность модели в распознанном тексте. В зависимости от уверенности присваивается одна из 3х категорий: sure, maybe, not sure. Если файл с таким именем уже существует добавляется _1, _2 и так далее.
